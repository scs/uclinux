Blackfin Network Scope Demo Overview

The Network Scope, demonstrates a simple mechanism to share access and data distributed over a TCP/IP network. The web browser contacts the HTTP server running on the Blackfin processor where the CGI script or backend program resides. Parameters from the HTML form are passed to the program in a similar manor as parameters are passed through an Environment, or to the command line.

The called program samples data from the ADC using a linux device driver. Incoming data samples are processed and stored in a data file. The backend CGI program then calls Gnuplot and requests to generate a PNG or JPEG image based on the sampled data and form settings. (Gnuplot is a portable command-line driven interactive datafile and function plotting utility). The server takes the output of the CGI program and returns it to the web browser which displays the output as an HTML page.

http://blackfin.uclinux.org/
Features:
  • Variable Sample Rate
  • Variable Sample Depth
  • Several Trigger Options (State, Edge, Variable Level)
  • Different Plotting and Displaying Options
  • Statistical Functions (Min, Max, Mean, etc.)
Platform:
  • Blackfin STAMP Board (ADDS-BF5XX-STAMP)
  • uClinux 2.6.x for Blackfin
  • AD7476A-DBRD
